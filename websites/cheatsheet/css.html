CSS Cheatsheet

Tag Name Description
#id Id grabber Grabs an element based on it's id to assign specific css
.attribute Attribute grabber Grabs an attribute based on it's tag to assign specific css
element Element grabber Grabs an element based on it's name to assign specific css
font-weight Font Weight Bolds font or normalizes font
font-size Font size Sets a font's size
